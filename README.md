# tsc-backoffice-automation

## CodeceptJS Setup
- npm init --yes
- npm install codeceptjs playwright --save or npm install codeceptjs playwright prettier --save
- npx codeceptjs init
- npm install --save-dev codeceptjs allure-codeceptjs 

## Creating a new page object
- npx codeceptjs gpo
